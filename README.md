### Getting Started

1 - Prerequisites

- **Node.js** installed on your machine.

2 - Installation

- Clone the repository:

```
    git clone https://github.com/claudiocaldeirao/js-data-structures-playground
    cd js-data-structures-playground
```

- Install dependencies:

```
    npm install
```

- Usage:

```
    npm start
```
